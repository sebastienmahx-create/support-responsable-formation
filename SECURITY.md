# Security Policy

## À propos

Cette ressource est fournie à titre éducatif pour la formation **BPJEPS Éducateur Sportif mention Activités de Judo-Jujitsu** (RNCP40424).

C'est une page web statique sans secrets, identifiants, ou données sensibles.

## Signaler une vulnérabilité

Si vous découvrez une vulnérabilité de sécurité dans cette ressource :

1. **Ne pas créer une issue publique** — Cela pourrait exposer la vulnérabilité
2. **Envoyer un email** à : sebastienmaheu@gmail.com (ou via GitHub Security Advisory)
3. **Inclure** :
   - Description de la vulnérabilité
   - Étapes pour la reproduire
   - Impact potentiel
   - Suggestions de correction (le cas échéant)

Vous recevrez une réponse dans les **48 heures**.
